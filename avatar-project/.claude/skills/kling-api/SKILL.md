# Kling AI API
Install: pip install pyjwt requests
Auth: JWT token from KLING_ACCESS_KEY + KLING_SECRET_KEY
Endpoint: https://api.klingai.com/v1/videos/image2video
Model: kling-v1-6
Token expires: 30 minutes, regenerate as needed
